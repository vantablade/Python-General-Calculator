# Python-General-Calculator
Personal Python project to test and develop my knowledge by incorporating many concepts to create a calculator for general purposes.

## Features
* Basic calculator: takes 2 numbers and an operator to calculate basic expressions
* BMI calculator: calculates your BMI and gives an approximation of your category of health
* Currency calculator: uses forex python to convert between several currencies in realtime prices
* Interest calculator: can calculate simple and compound interest

## Technologies Used
* Python 3
* [forex-python](https://pypi.org/project/forex-python/)

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/vantablade/general-calculator.git
```

### 2. Navigate into the project folder

```bash
cd general-calculator
```

### 3. Install required dependencies

```bash
pip install forex-python
```

## Running the Program

```bash
python calculator.py
```
